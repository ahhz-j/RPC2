# RPC2

Raid Player Counter AddOn for WoW Classic + Titan Panel.

## Installation

1. Copy `RPC2` into your `World of Warcraft/_classic_/Interface/AddOns/` folder.
2. Ensure Titan Panel is installed and enabled.
3. Reload UI (`/reload`) and enable `RPC2` in Titan's plugin list.

## What it shows

- `Raid`: current raid member count
- `Party`: current party member count
- `Solo`: `1`
