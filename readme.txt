RPC (Raid Player Count)
=============================================================================
版本: 0.1.7
适配版本: World of Warcraft: WotLK Classic

【插件简介】
RPC 是一款专为巫妖王之怒（WotLK）怀旧服设计的团队配置分析与监控插件。它能够自动检测团队成员的天赋专精和职责，并根据预设的团队模板实时显示配置达成情况，帮助团长快速补齐团队短板。

【核心功能】

1. 智能专精检测
   - 混合识别机制：结合 `Inspect` (观察) API 与光环/Buff 启发式检测算法。
   - 无需全员安装：即使队友未安装本插件，也能通过分析其身上的 Buff（如暗影形态、枭兽光环、图腾等）或能量类型准确识别专精。
   - 调试模式：提供详细的识别逻辑日志，方便排查识别错误。

2. 团队配置模板
   - 25人保底配置 (CCS10)：
     监控 10 个关键的团队增益覆盖情况，包括嗜血/英勇、法术急速、物理易伤、回蓝效果等核心 Buff 提供者。
   - 23人宝库推荐配置 (CS23)：
     针对阿尔卡冯的宝库（冬拥湖副本）设计的 23-25 人组队模板，确保涵盖各职业/护甲类型的 Tank、治疗和 DPS，优化装备分配效率。

3. 实时状态监控
   - 职业统计：实时显示各职业在线人数。
   - 职责统计：自动统计当前的 坦克 (Tank) / 治疗 (Healer) / 输出 (DPS) 数量。
   - 状态监控：实时显示团队成员的 存活、死亡、暂离 (AFK)、离线 状态。

4. 可视化界面
   - 图标指示：满足配置要求的图标会点亮，鼠标悬停可查看提供该 Buff/职责的具体玩家名单。
   - 缺失提示：未满足的配置显示为灰色，方便快速定位缺失职业。

【使用命令】
- /rpc
  打开或关闭插件主窗口。

【安装说明】
1. 将下载的 `RPC` 文件夹解压至魔兽世界安装目录下的 `Interface\AddOns` 文件夹中。
2. 目录结构应为：
   `...\Interface\AddOns\RPC\RPC.toc`
   `...\Interface\AddOns\RPC\RPC.lua`
   `...\Interface\AddOns\RPC\data.lua`
3. 登录游戏，在角色选择界面确认勾选 "RPC (Raid Player Count)"。
